# **App Name**: EduPulse

## Core Features:

- Administrative Dashboard: A high-level overview visualizing total enrollment, gender splits (Male/Female), grade-level (std) distribution, and age demographics using interactive charts.
- Student Lifecycle Manager: Complete management of student records including personal details, academic standard, age, and gender tracking.
- Demographic Insight Tool: An AI-powered tool that analyzes student distribution patterns to suggest optimal resource allocation or classroom balance across standards.
- Intelligent Data Entry: Leverage AI to process student information and automatically categorize records by age and standard from bulk text or CSV imports.
- Grade & Age Filters: Fast filtering system to view specific subsets of the student population by gender, age bracket, or academic standard.
- Centralized Records Database: Secure storage for all institutional data ensuring consistent reporting across all dashboard metrics.

## Style Guidelines:

- The palette is based on 'Scholarly Authority'. Primary color: Deep Cobalt (#2E5BDB) representing reliability; Background: Clean Cloud (#F3F5F9) for focus; Accent: Cyan Azure (#1DB6D8) for highlights.
- Headline and Stat font: 'Space Grotesk' (sans-serif) for its scientific, precise look; Body font: 'Inter' (sans-serif) for high legibility in dense data tables.
- Monochrome, thin-stroke SVG icons for clean navigational cues and demographic representation.
- A grid-based dashboard layout utilizing spacious cards and sidebars to minimize cognitive load.
- Progressive chart drawing and smooth lateral transitions between dashboard views to emphasize real-time data flow.
